## Key Features

### 1. Auto-Prompt Generation âœ¨
- **Automatically fills** the prompt field when modal opens
- **Includes**:
  - Day title
  - Day description (truncated to 200 chars if needed)
  - First 3 activities with descriptions
  - Context: "4:3 aspect ratio, suitable for a travel itinerary"

### 2. Edit Capability âœï¸
- Users can **modify** the auto-generated prompt
- Helpful for:
  - Adding specific details
  - Changing scene preferences
  - Adjusting tone/style

### 3. 4:3 Aspect Ratio ğŸ“
- Images generated in **4:3 ratio**
- Matches PDF display format
- Ensures consistent layout in tour documents

### 4. Dual Location ğŸ¯
- Available for **Day Images** (at itinerary day level)
- Available for **Activity Images** (within each activity)
